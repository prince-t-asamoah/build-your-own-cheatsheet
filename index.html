Semantic HTML Reference

Tag Name Description
<header> Header Defines a container for introductory content
<nav> Navigation Defines a container for set of navigational links
<main> Main Defines a container that holds most important content
<footer> Footer Defines a container for footer information such as copyright and authors name
<article> Article Defines a container for an independent, self-contained content
<section> Section Defines a container for a segment or part of a document
<aside> Aside Defines a container for contents not part of a main content or article
<figure> Figure Defines a container for self-contained contents such as illustrations, diagrams, photos, code listings etc.
<figcaption> Figure Caption Defines a title or description for figure contents
<details> Details Defines a piece of information that a user can show or hide
<summary> Summary Defines a heading for a detail content
<mark> Mark Defines a hightlighted text
<time> Time Defines a date and time
<address> Address Defines a contact information for the author or owner of document or an article
<video> Video Defines or embed a video content in a document, such as a movie clip or other video streams.
<audio> Audio Defines or embed an audio content in a document, such as a music or other audio streams.
